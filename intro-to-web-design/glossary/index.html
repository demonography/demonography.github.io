Glossary of Terms

Commonly used terminology and concepts introduced in Intro to Web Design.

UX

Short for User Experience - encompasses the entire experience of the user's interaction with a digital product, and their expectations and frustrations.

UI

Short for User Interface or also used for User Interface Design - how a user interacts with a product.

Persona

A summary of many people as one fictional person, from data gathered in research and discovery.

User

The person who is using and interacting with your product.

IA

Short for Information Architecture - the structure for a digital product that allows the user to know where they are and where the information is in relation to their position.

CMS

Short for Content Management System - an application or program created to manage digital content. Examples of popular CMS platforms are Wordpress and Squarespace.

CSS

Stands for: Cascading Style Sheets

CSS is made up of blocks, which are rules to tell the HTML what to do. Selectors are used to tell the CSS which HTML elements to target.

selector { property: value; }
body { background-color: blue; }
Turns the background-color of the body of the page (usually the whole page) blue.
p { color: red; }
Turns text in all paragraphs in the page red.
p { font-family: arial; }
Changes the font of the text in all paragraphs from the default computer font into Arial.
.custom { font-size: 30px; }
Remember the custom class we put on the p tag above? In css, we can target classes using a period before the name of the class we created (no space) and now only elements with the "custom" class will have a font size of 30px.
